This script is a cybersecurity tool that has been designed with the purpose of stealing all the information and the files of a computer using an USB device.

This tool has been created and designed by Pablo Corbalan (@HomeomorphicHooligan in GitHub).

For running this application, you should use:

  python script.py [options]

The optional options of the application are:

  -h: Displays helpt about the app
  -l: Displays the license of this application.
  -p <path>: The path to steal the files from by default it's the root.
  -o <output>: The output file, by default it's the path. 
  -rw: Removes writting permissions.
